# Симуляція Мобільного Робота (Курсова Робота)

Проект реалізує 2D симуляцію руху робота в обмеженому середовищі з перешкодами.
Використовується архітектура MVC, OpenGL для рендерингу та ImGui для інтерфейсу користувача.

## Функціонал

1. **Фізика руху:** Робот рухається з заданою швидкістю, обробляються зіткнення (Circl-AABB, Circle-Circle).
2. **Візуалізація:** - Відображення перешкод (кола, прямокутники).
   - Відображення точки зіткнення.
   - Відображення вектора швидкості.
3. **Керування:**
   - `W`, `A`, `S`, `D` - рух робота.
   - `R` - миттєве повернення на стартову позицію.
4. **Конфігурація:** Всі параметри сцени завантажуються з файлу `config.cfg`.
5. **UI (ImGui):** Панель для зміни швидкості, радіусу та перезавантаження сцени у реальному часі.

## Як запустити

1. Переконайтеся, що встановлено CMake та компілятор C++.
2. Створіть папку `build`:
   ```bash
   mkdir build
   cd build
   ```
3. Зберіть проект:
   ```bash
   cmake ..
   cmake --build . --config Release
   ```
4. Запустіть файл `.exe`. Переконайтеся, що `config.cfg` та папка `src/shaders` доступні програмі (лежать поруч або в корені проекту).
